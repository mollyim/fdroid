Molly d yilem n Signal i yettwaglen i tɣellist, i yettwasenker i yiseqdacen n Android. Tettmuddu timahilin-nniḍen yettwasɣawsenen, mgarad s iqeddacen n yizen n Signal.

Molly tessubur-d aḥraz s wawal uffir i taffa n yizen-inek d tidet, teddun daɣen d timahilin nniḍen am tallalt n yibenkan imeṛṛa d wayen-nniḍen. Tzemreḍ ad tferneḍ gar Molly neɣ Molly-FOSS, i d-ilelliwin d yeldin i wakken ad tferneḍ ma tebɣiḍ s yitran yettwaglen neɣ war-iten. Molly tettaleq Signal tamezwert war ma tesɛiḍ i tɣellist, u tettunefk i usnulfu n tɣuri.

<strong>Timamlayin igerrzen:</strong>

- Awgelhen n yisefka deg uɛeddi: ḥrez taffa n yizen-inek s uwgelhen n wawal uffir iɣef.
- Asfaɣer n RAM iɣellid: Sfeḍ s tɣellist isefka nniḍen seg takatut n yibenkan.
- Asekkar awurman n usnas: Sekker asnas deg yimeẓla uslig i tɣellist ufrin.
- Tallalt n yibenkan imeṛṛa: Sseddu yibenkan imeṛṛa, am tlifunat Android, ɣer umiḍan yiwen.
- Sewḥel imeskar ur nessinen ara: Ḥbes izen d tiliwin seg wid ur tessineḍ ara i tɣellist d ukensas.
- Amezruy n tiliwin yettḥebsen: Sfeḍ ilɣa n tiliwin akked izen yettwakkes.
- Iɣewwaren n uselkin n debug d ufrin: Sens aɣmis n Android i tɣellist.
- Asɣiwel n wadeg n uḥraz: Fren gar yal ass neɣ yal imalas akked amḍan n iḥrazen i tebɣiḍ.
- Tallalt n SOCKS proxy akked Tor: Ẓer abrid n uẓeṭṭa n usnas s proxy akked Orbot.
