Molly d amḍan n Signal i d-tṛuḥ amaynut, iḍelliḍen i wussan Android. Ttawiḍ iferdisn n tazwara s n Signal.

Molly yercḥ-ed tazwara s uɣilas n tazwara n ubrid nniḍen akk d iferdisn ameqqran iḍelliḍen, am tasnawt n taddart-nniḍen, d ayen nnid. Swaṭḥeḍ di Molly neɣ Molly-FOSS, ynna d iqṛiwan d ighlalen, s neɣ ur s-iḥseb ara i lqim n wumiḍen n ḥaḍṛin. Molly ɣermed Signal app ur tḥellef ara tazwara d amḍan d i d-yufaḍ ittwaclit.

<Strong>Aḥric n iferdisn:</Strong>

- Taḥfinit n iḍura deg yisr: Ḍfen tazwara-nwen s uḥrig n passphrase ameqran.
- Amḍan n RAM i s-sel: Ḍfen iḍura n wul i d-ssin deg memory n l-app s uḥrig.
- Tqiṭṭ n l-app ɣer-s : Qṭṭ l-app deg tmeddurt nniḍen i tḥelluft.
- Tasnawt n taddart-nniḍen: Sɛmed iḍura n taddart, am tablet n Android, ɣer account yiwen.
- Sakkar n iḍurar ur neqqar ara: Ɣef tazwara d tili n imḍurar ur neqqar ara s uḥrig d anti-spam.
- Agdud n tsnufft n tili: Sef l-bulun n tili s tazwara nnid.
- Dital n debug i s-sel: Sakkar logging n Android i uḍrim.
- Isemnawen n backup i tteddu: Swaṭḥeḍ deg ussan neɣ asbuɛ, akked umḍi n backup i ttwafaqen.
- SOCKS proxy akked Tor: Sekcem traffic n network n app s proxy akked Orbot.
