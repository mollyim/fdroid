Molly — це версія Signal, орієнтована на безпеку, покращена для користувачів Android. Вона пропонує додаткові функції при підключенні до серверів обміну повідомленнями Signal.

Molly повертає захист паролем для вашої локальної бази повідомлень і включає унікальні функції, такі як підтримка кількох пристроїв та інше. Вибирайте між Molly або Molly-FOSS, обидва безкоштовні та з відкритим кодом, з пропрієтарними компонентами або без них, залежно від ваших вподобань. Molly покращує стандартний додаток Signal, не шкодячи безпеці, і повністю відтворюваний.

<Strong>Ключові функції:</Strong>

- Шифрування даних у спокої: Захистіть базу повідомлень надійним паролем.
- Безпечне очищення RAM: Безпечно видаліть конфіденційні дані з пам’яті пристрою.
- Автоматичне блокування додатка: Заблокуйте додаток за певних умов для підвищеної безпеки.
- Підтримка кількох пристроїв: Підключайте кілька пристроїв, включно з планшетами Android, до одного акаунта.
- Блокування невідомих контактів: Запобігайте повідомленням та дзвінкам від невідомих відправників для безпеки та антиспаму.
- Зникаюча історія дзвінків: Очищайте сповіщення про дзвінки разом із терміновими повідомленнями.
- Додаткові журнали відладки: Вимкніть ведення журналів Android для конфіденційності.
- Кастомне планування резервних копій: Оберіть щоденні або щотижневі інтервали та кількість резервних копій для збереження.
- Підтримка SOCKS proxy та Tor: Направляйте мережевий трафік додатка через proxy та Orbot.

Для оновлень, новин та підтримки слідкуйте за @mollyim@fosstodon.org. Повідомляйте про помилки та запити функцій на сторінці GitHub проєкту.
