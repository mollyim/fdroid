Molly 是針對安全的 Signal 增強版，適用於 Android 使用者。在連接 Signal 訊息伺服器的同時，提供額外功能。

Molly 恢復了本地訊息資料庫的密碼保護，並包含獨特功能，如多裝置支援等。可在 Molly 或 Molly-FOSS 之間選擇，兩者均為免費開源，可依偏好選擇是否包含專有組件。Molly 在不影響安全性的情況下改進了原版 Signal 應用，並且完全可重現。

<Strong>主要功能:</Strong>

- 靜態資料加密：以強密碼保護訊息資料庫。
- 安全 RAM 清理：安全刪除裝置記憶體中的敏感資料。
- 自動應用鎖：在特定條件下鎖定應用以增強安全性。
- 多裝置支援：將包括 Android 平板在內的多個裝置連結至同一帳戶。
- 阻止未知聯絡人：防止未知發件人的訊息和通話。
- 消失的通話紀錄：連同過期訊息一併清除通話通知。
- 可選除錯日誌：停用 Android 日誌以保護隱私。
- 自訂備份計劃：選擇每日或每週的間隔及保留的備份數量。
- SOCKS 代理及 Tor 支援：透過代理及 Orbot 路由應用程式網路流量。

有關更新、新聞及支援，請關注 @mollyim@fosstodon.org。在專案 GitHub 頁面提交錯誤及功能請求。
