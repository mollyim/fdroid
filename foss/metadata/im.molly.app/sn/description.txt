Molly ishiyo yakatarisana nekuchengetedzwa, yakawedzerwa shanduro yeSignal yevashandisi veAndroid. Inopa zvimwe zvimwe zvinhu panguva yekubatana kune maseva eSignal ekutumira mameseji.

Molly inodzosa kuchengetedzwa kwepassword kune yako local message database uye inosanganisira zvinhu zvakasiyana-siyana zvakaita sekutsigira midziyo yakawanda, nezvimwe. Sarudza pakati peMolly kana Molly-FOSS, zvese zviri zviviri zviri mahara uye open-source, pamwe kana pasina proprietary blobs, zvichienderana nezvaunoda. Molly inovandudza stock Signal app pasina kukanganisa chengetedzo uye inogadziriswa zvizere.

<Strong>Zvakakosha Zvimiro:</Strong>

- Encryption yedata iri kuchengetedzwa: Chengetedza database yemeseji yako ne password yakasimba.
- Secure RAM wiper: Bvisa data yakakosha kubva kumemori yechigadzirwa zvakachengeteka.
- Automatic app lock: Vhara app pasi pemamiriro akasarudzwa kuti uwedzere kuchengetedzeka.
- Multi-device support: Batanidza midziyo yakawanda, kusanganisira Android mamatabhu, kune imwe account.
- Block unknown contacts: Dzivirira mameseji nemafoni kubva kune vasina kuzivikanwa vanotumira zvekuchengetedzeka uye anti-spam.
- Disappearing call history: Bvisa zvikumbiro zvekufona pamwe nemeseji dzinopera.
- Optional debug logs: Dzima Android logging kuti uve neprivacy.
- Custom backup scheduling: Sarudza mainterval ezuva nezuva kana pasvondo uye nhamba yemabackup ekuchengetedza.
- SOCKS proxy uye Tor support: Rerutsa network traffic yeapp kuburikidza neproxy uye Orbot.

Kuti uwane zvigadziriso, nhau, uye rutsigiro, tevera @mollyim@fosstodon.org. Tumira bugs uye zvikumbiro zvefeatures papeji reGitHub reprojekiti.
