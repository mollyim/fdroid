Use F-Droid to install apps and ensure access to app updates

F-Droid is an independent, community-sourced app store. Adding a repository to F-Droid makes the apps within it available for download and ensures that you have access to secure app updates.

Download F-Droid at f-droid.org.

Install Molly via F-Droid

1. Open this link on your mobile device, or scan the QR code.

The link should open in the F-Droid app.

2. When the Add new repository screen appears, click Add.

3. F-Droid will update the list of available apps. When it finishes, search for Molly.
