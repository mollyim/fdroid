Molly es una versión mejorada y centrada en la seguridad de Signal para usuarios de Android. Ofrece funciones adicionales mientras se conecta a los servidores de mensajería de Signal.

Molly recupera la protección mediante frase de contraseña para tu base de datos local de mensajes e incluye funciones únicas como el soporte multidispositivo y más. Elige entre Molly o Molly-FOSS, ambas opciones gratuitas y de código abierto, con o sin componentes propietarios según tus preferencias. Molly mejora la aplicación Signal original sin comprometer la seguridad y es completamente reproducible.

<Strong>Características clave:</Strong>

- Cifrado de datos en reposo: Protege tu base de datos de mensajes con un cifrado robusto basado en frase de contraseña.
- Limpiador seguro de RAM: Borra de forma segura los datos sensibles de la memoria del dispositivo.
- Bloqueo automático de la app: Bloquea la aplicación bajo condiciones específicas para mayor seguridad.
- Soporte multidispositivo: Vincula varios dispositivos, incluidas tabletas Android, a una sola cuenta.
- Bloqueo de contactos desconocidos: Evita mensajes y llamadas de remitentes desconocidos.
- Historial de llamadas que desaparece: Elimina las notificaciones de llamadas junto con los mensajes que expiran.
- Registros de depuración opcionales: Desactiva el registro de Android para proteger la privacidad.
- Programación personalizada de copias de seguridad: Elige intervalos diarios o semanales y el número de copias a conservar.
- Compatibilidad con proxy SOCKS y Tor: Canaliza el tráfico de red de la app a través de un proxy y Orbot.
