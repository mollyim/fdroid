Molly 係專注安全、加強版嘅 Signal，適用於 Android 用戶。免費、開源，而且冇任何專有軟件。佢提供額外功能，同時連接到 Signal 嘅訊息伺服器。  

Molly 為你嘅本地訊息數據庫帶返密碼保護，並包括獨特功能，好似多裝置支援等。Molly 喺唔犧牲安全嘅情況下改善咗原版 Signal 應用，而且完全可重現。  

<Strong>主要功能：</Strong>

- 靜態數據加密：用強密碼加密保護你嘅訊息數據庫。  
- 安全清除 RAM：安全地清除裝置記憶體入面嘅敏感資料。  
- 自動鎖定應用：喺特定情況下鎖定應用以增加安全性。  
- 多裝置支援：將多部裝置，包括 Android 平板，連接到同一個帳號。  
- 阻止未知聯絡人：防止來自未知發件人嘅訊息同電話，以保障安全同防垃圾訊息。  
- 消失嘅通話歷史：連同過期訊息一齊清除通話通知。  
- 可選調試日誌：為咗私隱可以關閉 Android 記錄。  
- 自訂備份排程：選擇每日或每週間隔，同設定保留備份嘅數量。  
- 支援 SOCKS 代理同 Tor：透過代理同 Orbot 轉發應用網絡流量。  
- 通知：可以揀開源、冇專有軟件嘅 FCM、UnifiedPush 或 WebSocket 實現。  

想獲取更新、新聞同支援，可以 follow @mollyim@fosstodon.org。喺項目 GitHub 頁面提交 bug 同功能需求。
