Molly 係一個專注於安全嘅 Signal 增強版本，適用於 Android 用戶。佢喺連接 Signal 嘅訊息伺服器時提供額外功能。

Molly 為你嘅本地訊息數據庫帶返密碼保護，仲包括獨特功能，例如多設備支持等等。你可以揀 Molly 或 Molly-FOSS，兩個都係免費同開源選擇，可以有或者無專有組件，視乎你嘅喜好。Molly 喺唔影響安全嘅情況下改進咗 Signal 官方應用，並且可以完全重現。

<Strong>主要功能:</Strong>

- 靜態數據加密: 用強密碼加密保護你嘅訊息數據庫。
- 安全 RAM 清除: 安全地刪除設備內存中敏感數據。
- 自動應用鎖: 喺特定條件下鎖定應用以增強安全。
- 多設備支持: 將多個設備，包括 Android 平板，連接到同一帳號。
- 阻止未知聯絡人: 防止來自未知發送者嘅訊息同通話，以保障安全同防垃圾信息。
- 消失嘅通話記錄: 清除通話通知同到期訊息。
- 可選除錯日誌: 為私隱禁用 Android 記錄。
- 自訂備份排程: 選擇每日或每週間隔，同保留備份數量。
- SOCKS 代理同 Tor 支持: 透過代理同 Orbot 路由應用網絡流量。

要獲取更新、新聞同支援，跟蹤 @mollyim@fosstodon.org。喺項目 GitHub 頁面提交漏洞同功能請求。
