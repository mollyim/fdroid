Molly es una versión mejorada de Signal para usuarios de Android, enfocada en la seguridad. Ofrece características adicionales mientras se conecta a los servidores de mensajería de Signal.

Molly trae de vuelta la protección mediante frase de contraseña para tu base de datos local de mensajes e incluye funciones únicas como soporte multidispositivo y más. Podés elegir entre Molly o Molly-FOSS, ambas gratuitas y de código abierto, con o sin componentes propietarios según tus preferencias. Molly mejora la aplicación Signal original sin comprometer la seguridad y es totalmente reproducible.

<Strong>Funciones principales:</Strong>

- Cifrado de datos en reposo: Protegé tu base de datos de mensajes con cifrado fuerte de frase de contraseña.
- Limpiador seguro de RAM: Eliminá datos sensibles de la memoria del dispositivo de manera segura.
- Bloqueo automático de la app: Bloqueá la app bajo condiciones específicas para mayor seguridad.
- Soporte multidispositivo: Vinculá varios dispositivos, incluidas tablets Android, a una sola cuenta.
- Bloqueo de contactos desconocidos: Evitá mensajes y llamadas de remitentes desconocidos.
- Historial de llamadas que desaparece: Eliminá notificaciones de llamadas junto con mensajes que expiran.
- Registros de depuración opcionales: Desactivá el registro de Android para mayor privacidad.
- Programación personalizada de copias de seguridad: Elegí intervalos diarios o semanales y cuántas copias conservar.
- Compatibilidad con proxy SOCKS y Tor: Redirigí el tráfico de la app a través de un proxy y Orbot.

Para actualizaciones, noticias y soporte, seguí a @mollyim@fosstodon.org. Reportá errores y pedidos de funciones en la página de GitHub del proyecto.
