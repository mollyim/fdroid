Molly 係專注安全、增強版嘅 Signal，適用於 Android 用戶。免費、開源，並且唔含任何專有軟件。佢喺連接 Signal 嘅訊息伺服器時提供額外功能。  

Molly 為你嘅本地訊息資料庫帶返密碼保護，並包括獨特功能，例如多裝置支援等。Molly 喺唔犧牲安全性嘅情況下改進咗 Signal 官方應用，並且完全可重現。  

<Strong>主要功能：</Strong>

- 靜態資料加密：使用強密碼加密保護你嘅訊息資料庫。  
- 安全 RAM 清理：安全刪除裝置記憶體入面嘅敏感資料。  
- 自動應用鎖定：喺特定情況下鎖定應用以增強安全性。  
- 多裝置支援：將多部裝置，包括 Android 平板，連接到同一帳戶。  
- 阻止未知聯絡人：防止來自未知發送者嘅訊息同通話，保障安全並防垃圾訊息。  
- 消失嘅通話歷史：與過期訊息一齊清除通話通知。  
- 可選調試日誌：為咗私隱可以關閉 Android 記錄。  
- 自訂備份計劃：選擇每日或每週間隔，以及保留備份嘅數量。  
- 支援 SOCKS 代理同 Tor：透過代理同 Orbot 路由應用網絡流量。  
- 通知：可選擇開源、無專有軟件嘅 FCM、UnifiedPush 或 WebSocket 實現。  

如需更新、新聞及支援，請關注 @mollyim@fosstodon.org。喺項目 GitHub 頁面提交 bug 同功能請求。
